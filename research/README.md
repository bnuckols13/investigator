# Hesitation Media / Memory Hole — Research Repository

Investigative research produced by the OSINT Investigator Toolkit. Every finding is version-controlled, timestamped, and traceable to its primary source.

## Structure

- [scans/](./scans/) — Population-level nonprofit anomaly scans (dated snapshots)
- [cases/](./cases/) — Active investigation case files
- [methodology/](./methodology/) — Data sources, scoring systems, and protocols

## Methodology

This research uses the **Memory Hole Evidence Evaluation System (MHEES)** and the **Trustless Journalism Protocol**. Every claim carries an MHEES confidence code. Every data source is archived with SHA-256 hashes. See [methodology/](./methodology/) for complete documentation.

## Verification

All source data is archived locally and in multiple independent archives (Internet Archive, archive.ph). SHA-256 hashes are recorded for every download. Git commit history provides a tamper-evident audit trail.

To reproduce any analysis, clone this repository and follow the instructions in each scan's METHODOLOGY.md file.
